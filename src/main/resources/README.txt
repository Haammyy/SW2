    - Scheduling application -
------------------------------------------------------------------------------------

    This application is a JavaFX GUI designed to enable users to create, read, update,
    and delete customer and appointment records in a MySQL database.
    The application is designed for small business applications.
    The application allows users to view, add, update, and delete customers and appointments.
    The application also allows users to generate reports of customer and appointment data.

------------------------------------------------------------------------------------

    To run:
        install all libraries:
            - JavaFX-SDK-17.0.2
            - mysql-connector-java-8.0.31

        ensure that the database is running on port 3306, and that the login credentials are correct.

        Run the program from the Main class.

------------------------------------------------------------------------------------

    The additional report I chose is a table showing the appointments in order of when
    they are happening the soonest, this will provide crucial information at a glance.
